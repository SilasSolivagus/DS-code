# 2.5 Token 计数/估算对齐 CC — 实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 给 deepcode 补一个 CJK 感知的独立 token 估算模块 + 模型感知 context window + 发送前预估触发 compact，修中文低估并对齐 CC 阈值语义。

**Architecture:** 新建纯函数模块 `src/tokenEstimate.ts`（4 函数 + 常量），替换散落的 `chars/4`、`chars/3` 启发式；`compactTokens` 默认改 undefined 使「window−buffer 派生阈值」生效；useChat 用「与 lastPromptTokens 原子配对的 baselineLen + loop.ts 发送时捕获的 sentLen」做发送前增量预估；状态栏改显绝对值 `used / window`。

**Tech Stack:** TypeScript/ESM、vitest、ink5 TUI、OpenAI 兼容 SDK（DeepSeek）。

设计文档：`docs/specs/2026-06-19-deepcode-2.5-token-counting-design.md`（先读）。

## Global Constraints

- **官方比例**（已查实）：CJK ×0.6/字、其余字符 ×0.3/字（两分类启发式，over-estimate 偏安全）。
- **常量**：`CONTEXT_WINDOW_DEFAULT = 200_000`；模型表 `{ 'deepseek-v4-flash': 1_000_000, 'deepseek-v4-pro': 1_000_000 }`；`OUTPUT_RESERVE = 16_000`；`AUTOCOMPACT_BUFFER = 13_000`；env 覆盖键 `DEEPCODE_MAX_CONTEXT_TOKENS`。
- **派生阈值** = `resolveContextWindow(model) − OUTPUT_RESERVE − AUTOCOMPACT_BUFFER`（flash/pro = 971_000）。
- **`compactTokens` 语义**：未设（undefined）→ 走派生阈值；已设 → `min(派生, compactTokens)`（更紧省钱上限，向后兼容）。
- **baseline 不变量**：`baselineLen` 与每次 `lastPromptTokens` 写入原子配对；归零点 init/clear/compact/resume 设 `baselineLen=0`；turn_end 设 `baselineLen=ev.sentLen`（loop.ts 发送前捕获，**非** turn_end 时的 messages.length）；读取处一律 `slice(Math.min(baselineLen, messages.length))` 防越界。
- **测试命令**：`npx vitest run <file>`（项目用 vitest，`vitest.config.ts` include `test/**/*.test.{ts,tsx}`）。全量 `npm test`。typecheck `npm run build` 或 `npx tsc --noEmit`。
- **提交**：每任务一 commit，遵循既有中文 commit 风格。**禁止** `git add -A`（历史上误纳 AGENTS.md/.claude/）——只 `git add` 本任务明确文件。
- 所有 fail-safe 函数对空/undefined/未知输入返回兜底值，绝不抛。

```ts
// src/tokenEstimate.ts

/** 判断 code point 是否属 CJK 表意区段（中日韩统一表意 + 扩展 + 兼容 + 假名/谚文常用） */
function isCJK(cp: number): boolean {
  return (
    (cp >= 0x4e00 && cp <= 0x9fff) ||   // CJK 统一表意
    (cp >= 0x3400 && cp <= 0x4dbf) ||   // 扩展 A
    (cp >= 0x20000 && cp <= 0x2ebef) || // 扩展 B-F
    (cp >= 0xf900 && cp <= 0xfaff) ||   // 兼容表意
    (cp >= 0x3040 && cp <= 0x30ff) ||   // 平假名/片假名
    (cp >= 0xac00 && cp <= 0xd7af)      // 谚文音节
  )
}

/** CJK 感知 token 估算：CJK ×0.6/字、其余 ×0.3/字。空/undefined → 0，绝不抛。
 *  基于 DeepSeek 官方比例（中文 0.6、英文 0.3 token/字符）。over-estimate 偏安全。 */
export function estimateTextTokens(s: string | null | undefined): number {
  if (!s) return 0
  let weighted = 0
  for (const ch of s) {
    const cp = ch.codePointAt(0)!
    weighted += isCJK(cp) ? 0.6 : 0.3
  }
  return Math.ceil(weighted)
}
```

```ts
// 追加到 src/tokenEstimate.ts
/** 按 deepcode 扁平 OpenAI 消息结构逐条累加 token 估算。
 *  content 永远是 string|null；assistant.tool_calls[].function 的 name+arguments 计入。
 *  无 Anthropic block 数组、无图像分支（V4 纯文本）。整体一次 ceil 避免逐条 ceil 累积偏高。 */
export function estimateMessagesTokens(messages: any[]): number {
  let weighted = 0
  for (const m of messages ?? []) {
    weighted += rawWeight(typeof m?.content === 'string' ? m.content : '')
    if (Array.isArray(m?.tool_calls)) {
      for (const tc of m.tool_calls) {
        const fn = tc?.function ?? {}
        weighted += rawWeight((fn.name ?? '') + (fn.arguments ?? ''))
      }
    }
  }
  return Math.ceil(weighted)
}
```

并把 Task 1 的加权抽成共享 `rawWeight`（不 ceil 的内部纯函数），`estimateTextTokens` 改为 `Math.ceil(rawWeight(s))`：

```ts
function rawWeight(s: string | null | undefined): number {
  if (!s) return 0
  let w = 0
  for (const ch of s) { const cp = ch.codePointAt(0)!; w += isCJK(cp) ? 0.6 : 0.3 }
  return w
}
export function estimateTextTokens(s: string | null | undefined): number {
  return Math.ceil(rawWeight(s))
}
```

```ts
// 追加到 src/tokenEstimate.ts
export const CONTEXT_WINDOW_DEFAULT = 200_000
const MODEL_CONTEXT_WINDOWS: Record<string, number> = {
  'deepseek-v4-flash': 1_000_000,
  'deepseek-v4-pro': 1_000_000,
}
const OUTPUT_RESERVE = 16_000
const AUTOCOMPACT_BUFFER = 13_000

/** 模型感知 context window：env 覆盖 → 模型表 → 默认。未知模型回落默认，绝不抛。 */
export function resolveContextWindow(model: string): number {
  const env = process.env.DEEPCODE_MAX_CONTEXT_TOKENS
  if (env) { const n = parseInt(env, 10); if (Number.isFinite(n) && n > 0) return n }
  return MODEL_CONTEXT_WINDOWS[model] ?? CONTEXT_WINDOW_DEFAULT
}

/** compact 触发派生阈值 = window − 输出预留 − autocompact buffer。 */
export function computeCompactThreshold(model: string): number {
  return resolveContextWindow(model) - OUTPUT_RESERVE - AUTOCOMPACT_BUFFER
}
```
